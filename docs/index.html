Cloud Computing

Definición

Cloud computing es la entrega de diferentes servicios mediante el internet. Estos recursos incluyen herramientas y aplicaciones como almacenamiento de datos, servidores, base de datos, redes y software.

En lugar de guardar archivos en un disco duro o en un almacenamiento local, el almacenamiento basado en la nube hace posible guardarlos en una base de datos remota. Siempre y cuando un dispositivo electrónico tenga acceso a la red, tendrá acceso a los datos y al software para ejecutarlo.

Cloud computing es nombrado así porque la información que es accesada se encuentra remota en la nube o un espacio virtual. Compañías que proporcionan servicios de la nube permiten a los usuarios almacenar archivos y aplicaciones en servidores remotos y despues accesar todos los datos vía internet. Esto significa que no es necesario que el usuario este en un lugar especifico para tener acceso a esto, permitiendo al usuario a trabajar remotamente.

Cloud computing puede ser tanto publico como privado. Los servicios publicos proporcionan sus servicios por medio del internet por una tarifa. Los servicios privados por otra parte solo proveen servicios a un cierto número de personas. Hay tambien una opción hibrida, la cual combina elementos de ambos, tanto la publica como la privada.
